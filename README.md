# Health Tracker — Mobile App

A personal health tracking mobile application built with **React Native** and **Expo**, developed as part of a **Mobile App Development Challenge** inspired by a design mood board.

---

## Project Description

Health Tracker is a comprehensive mobile application that helps users monitor and improve their daily health habits. The app provides tools for tracking physical activity, blood sugar levels, breathing exercises, and personalized weekly wellness plans — all in one place. The project was developed as a submission for a mobile development challenge, combining functional health features with a clean and intuitive user interface.

---

## Features

- **User Authentication** — Register and login with Firebase Authentication
- **Profile Setup** — Personalized onboarding flow for new users
- **Dashboard (Home Screen)** — Overview of daily health status and quick actions
- **Activity Tracking** — Log and monitor physical activities
- **Blood Sugar Tracking** — Record and review blood sugar readings
- **Weekly Wellness Plan** — AI-assisted personalized weekly plan generation
- **Breathing Exercises** — Guided breathing exercise screen
- **Insights & Statistics** — Visual charts and progress overview
- **Daily Goals** — Set and track personal health goals
- **Risk Survey** — Health risk assessment questionnaire
- **AI Assistant** — In-app AI support screen
- **FAQ Screen** — Frequently asked questions with accordion UI
- **Push Notifications** — Reminders and health alerts via Expo Notifications
- **Bottom Tab Navigation** — Smooth navigation between main sections
- **Responsive Mobile UI** — Optimized for both Android and iOS

## How to Run the Project Locally

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Expo Go](https://expo.dev/go) app installed on your mobile device (Android or iOS)

### Steps

**1. Clone the repository**

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npx expo start
```

**4. Open on your device**

- A QR code will appear in the terminal
- Open the **Expo Go** app on your phone
- Scan the QR code to launch the application

> **Note:** Make sure your phone and computer are connected to the same Wi-Fi network.

---

## Expo Access

The application can be run directly on any physical device using the **Expo Go** app — no build or installation required.

1. Download **Expo Go** from the [App Store](https://apps.apple.com/app/expo-go/id982107779) or [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)
2. Start the project with `npx expo start`
3. Scan the QR code shown in the terminal with Expo Go
