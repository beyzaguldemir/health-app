## Assets Klasörü

Bu klasöre aşağıdaki dosyaları eklemeniz gerekir:

### Gerekli Dosyalar:

1. **icon.png** (1024x1024 px) - Uygulama ikonu
2. **splash.png** (1284x2778 px) - Açılış ekranı
3. **adaptive-icon.png** (1024x1024 px) - Android adaptive icon
4. **notification-icon.png** (96x96 px) - Bildirim ikonu (Android)

### Hızlı Çözüm:

Geçici olarak placeholder görseller oluşturmak için:

```bash
# Basit renkli kareler oluşturabilir veya
# https://www.favicon.cc/ gibi sitelerden ikon oluşturabilirsiniz
```

### Icon Generator Siteleri:

- https://www.appicon.co/
- https://makeappicon.com/
- https://icon.kitchen/

Bu dosyalar olmadan da uygulama çalışır, ancak varsayılan Expo ikonları görünür.
